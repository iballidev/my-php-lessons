# ACCESSING MySQL IN THE COMMAND LINE INTERFACE (XAMPP server)
https://stackoverflow.com/questions/5920136/mysql-is-not-recognised-as-an-internal-or-external-command-operable-program-or-b

1. Locate "C:\xampp\mysql" folder

2. Access it's path with the command line interface:

        C:\xampp\mysql>

3. Attempt to access mysql (it will fail):

        C:\xampp\mysql>mysql -u root --password=swordfish01
        ERROR 1045 (28000): Access denied for user 'root'@'localhost' (using password: YES)

4. Access the bin file path:

        C:\xampp\mysql>path
       PATH=C:\xampp\mysql\bin

5. Set the bin file path:
        C:\xampp\mysql>set PATH=C:\xampp\mysql\bin

6. Enter empty password:
        C:\xampp\mysql>mysql -u root --password
        Enter password:
        NOTE: Do not enter any text for password just press enter again

        
        Welcome to the MariaDB monitor.  Commands end with ; or \g.

# MySQL DATABASE

1. Create a MySQL Database: 

        MariaDB [(none)]> CREATE DATABASE my_test_db;        
        Query OK, 1 row affected (0.004 sec)

2. Access the database:
        
        MariaDB [(none)]> use my_test_db
        Database changed

3. Create a table "subjects" inside the database:

        MariaDB [my_test_db]> CREATE TABLE subjects(
            -> id int(11) NOT NULL auto_increment,
            -> menu_name varchar(30) NOT NULL,
            -> position int(3) NOT NULL,
            -> visible tinyint(1) NOT NULL,
            -> PRIMARY KEY (id)
            -> );
        Query OK, 0 rows affected (0.039 sec)

4. Read the table data:
        
        MariaDB [my_test_db]> select * from subjects;
        Empty set (0.096 sec)

        NOTE: The table is currently empty because no data has been added into it.

5. Insert data into the table:

        MariaDB [my_test_db]> INSERT INTO subjects(menu_name, position, visible)
            -> VALUES('About Widget Corp', 1, 1);
        Query OK, 1 row affected (0.004 sec)
        

6. Read the table data again:

        MariaDB [my_test_db]> select * from subjects;
